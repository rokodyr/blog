# Cyber Security Roadmap

This roadmap outlines a structured learning path for anyone interested
in **cyber security**, from fundamentals to advanced offensive security.

---

## 1. Basics of Cyber Security

- Computer Networking Fundamentals
  - TCP/IP, UDP, HTTP/S
  - DNS, DHCP, Routing
- Operating Systems Basics
  - Windows, Linux fundamentals
  - File systems, permissions
- Basic Security Concepts
  - CIA Triad (Confidentiality, Integrity, Availability)
  - Authentication & Authorization
  - Encryption fundamentals

---

## 2. Intermediate Skills

- System Internals
  - Processes, memory, services
  - Kernel vs User mode
- Programming & Scripting
  - Python, Bash, PowerShell
  - C/C++ for low-level understanding
- Networking & Protocol Analysis
  - Packet capturing & analysis (Wireshark)
  - Firewalls, NAT, routing security
- Web Security Basics
  - HTTP methods, headers
  - Common vulnerabilities (XSS, SQLi, CSRF)
