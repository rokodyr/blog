# Offensive OSINT

## Overview
OSINT (Open Source Intelligence) is about gathering information safely
to analyze systems and human factors.

## Topics
- Difference between information and actionable intelligence
- Data filtering: noise vs signal
- Social media analysis and profiling
- Human behavior analysis for security context
- System reconnaissance: services, network, configurations

## Takeaway
Effective OSINT requires critical thinking, not just tool usage.
